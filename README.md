# Terminal Forest – C++ Console Game
Terminal Forest is a text-based, grid-style adventure game built in C++ for the Windows console.  
You explore a hostile forest, fight enemies, collect items and keys, and progress through multiple levels to escape.

---

## Features
- Pure C++ console application (no external engines)
- Turn-based combat with different enemy behaviours
- Items and buffs that affect player stats
- Key/door system for progression between areas
- Multiple levels with increasing difficulty
- Save/Load system using a central GameData module

---

## Controls
- **W / A / S / D** – Move up, left, down, and right  
- **K** – Save game  
- **Q** – Quit to main menu (if implemented)

Map and HUD information are displayed directly in the console.

---

## Building & Running
1. Open the solution (`.sln`) in **Visual Studio** (or another IDE).
2. Make sure the configuration is set to **Debug** or **Release**, **x86/x64** as appropriate.
3. Build the solution.
4. Run the resulting executable from within the IDE.
